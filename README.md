# üìù Todo List App

A modern **React Native Todo List App** that allows users to create, manage, and track their daily tasks easily.  
This app features **light and dark mode**, integrates **Convex** as a backend for data persistence, and includes task filtering options for a better user experience.

---

##  Features

-  **Add Todo:** Create new tasks easily using the input field.
-  **Mark as Completed:** Tap the check icon to mark tasks as done.
-  **Delete Todo:** Remove unwanted tasks from the list.
-  **Convex Backend Integration:** All todos are stored and managed with Convex, so data remains even after closing the app.
-  **Light & Dark Mode:** Toggle between light and dark themes for a personalized experience.
-  **Filter Options:**
  - **All:** View every task
  - **Active:** View only pending tasks
  - **Completed:** View completed tasks
